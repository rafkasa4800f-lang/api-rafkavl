Brat API Gabungan

Ini satu project lengkap: tampilan web, endpoint API, tester URL, contoh kode plugin bot, dan konsep timpa/banyak frame. Warna cukup pakai kode HEX, bukan preset.

Web tester GET /api/brat POST /api/brat Plugin bot via API Image-only tanpa bratvid

Pakai langsung dari web

Contoh: #ff0000, #b266ff, #1e90ff, #f4a2b9

URL API

Bagian ini buat tester request API. Output URL bisa dipakai di bot, browser, atau fetch.

Konsep timpa/banyak-banyak

          
Bratvid tidak dibuat di Vercel. API hanya membuat PNG. Bot/VPS nanti panggil API berkali-kali untuk membuat frame.

Cara deploy ke GitHub + Vercel

        

          
1. Upload semua isi folder project ini ke repository GitHub.

          
2. Buka Vercel, pilih Add New Project, lalu import repository tadi.

          
3. Framework preset boleh Other. Build command boleh kosong.

          
4. Setelah deploy, buka domain Vercel kamu. Halaman ini akan muncul sebagai web tester.

          
5. Endpoint API aktif di /api/brat?text=halo&color=ff0000.
